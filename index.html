Problema de Ingeniería Civil

Solución del sistema de ecuaciones para mezcla de materiales de canteras

Planteamiento del Problema

Un ingeniero civil requiere los siguientes materiales para una obra:

  • 4,800 m³ de arena
  • 5,810 m³ de grano fino
  • 5,690 m³ de grano grueso

Existen tres canteras con diferentes composiciones de materiales:

Cantera Arena (%) Grano Fino (%) Grano Grueso (%)
Cantera 1 52% 30% 18%
Cantera 2 20% 50% 30%
Cantera 3 25% 20% 55%

Se necesita determinar cuántos metros cúbicos se deben extraer de cada cantera para cumplir con los requerimientos.

Sistema de Ecuaciones

Definimos las variables:

  • x₁: metros cúbicos de Cantera 1
  • x₂: metros cúbicos de Cantera 2
  • x₃: metros cúbicos de Cantera 3

El sistema de ecuaciones queda:

0.52x₁ + 0.20x₂ + 0.25x₃ = 4800

0.30x₁ + 0.50x₂ + 0.20x₃ = 5810

0.18x₁ + 0.30x₂ + 0.55x₃ = 5690

Multiplicando por 100 para eliminar decimales:

52x₁ + 20x₂ + 25x₃ = 480,000

30x₁ + 50x₂ + 20x₃ = 581,000

18x₁ + 30x₂ + 55x₃ = 569,000

Solución del Sistema

Resolviendo el sistema mediante el método de matrices:

Resultados:

Cantera 1 (x₁): 4,011.63 m³

Cantera 2 (x₂): 7,162.79 m³

Cantera 3 (x₃): 5,125.58 m³

Verificación:

Arena: 0.52×4011.63 + 0.20×7162.79 + 0.25×5125.58 = 4,800 m³ ✓

Grano fino: 0.30×4011.63 + 0.50×7162.79 + 0.20×5125.58 = 5,810 m³ ✓

Grano grueso: 0.18×4011.63 + 0.30×7162.79 + 0.55×5125.58 = 5,690 m³ ✓

Total de material extraído: 16,300 m³

Métodos de Solución

Método de Matrices

Se utiliza la fórmula X = A⁻¹B, donde:

  • A es la matriz de coeficientes
  • B es el vector de términos independientes
  • X es el vector solución

Método de Eliminación de Gauss

Se transforma la matriz aumentada [A|B] a forma escalonada mediante operaciones elementales de fila.

Método de Jacobi

Método iterativo que puede aplicarse cuando la matriz es diagonalmente dominante.

Aplicación en Ingeniería Civil

Este tipo de problemas es común en:

  • Optimización de recursos en construcción
  • Mezcla de materiales para hormigón
  • Planificación de extracción en canteras
  • Minimización de costos de transporte

La solución obtenida permite al ingeniero:

  • Planificar la logística de extracción
  • Optimizar los costos de transporte
  • Cumplir con las especificaciones técnicas
  • Minimizar el desperdicio de materiales
